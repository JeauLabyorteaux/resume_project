Copy 3D, Paste Object [C3PO]

University of Nevada, Reno | Capstone Senior Project

Overview

For my Senior Capstone project at the University of Nevada, Reno, I collaborated with a team to develop C3PO, an end-to-end 3D scanning and reconstruction system. Our primary objective was to bridge the gap between physical objects and digital 3D models by creating a streamlined, user-friendly "plug-and-play" hardware and software solution.

Technical Implementation

The system leverages the Intel RealSense D405 depth camera to capture high-fidelity spatial data. This data is passed through a custom rendering pipeline designed to process raw point-cloud data into 3D-printable files.

Key Achievements

  • Designed a streamlined UI with interactive.
  • Integrated hardware-software communication to allow for automated software installation upon device connection.
  • Developed a processing pipeline to convert sensor data into accurate, manifold meshes suitable for additive manufacturing.
